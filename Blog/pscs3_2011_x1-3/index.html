仿制图章

①②


使用 仿制图章工具 在左上角修改画笔 大小 和 硬度 ，分别是 16 和 50

按住 Alt 使用 鼠标左键 选择较为中间的位置 之后鼠标上移 对齐图形 再使用 鼠标左键 覆盖

在使用过程中会发现在覆盖时后面会跟着一个 十字光标 ，这个光标代表你在用哪部分的图形覆盖，重置它位置的方法很简单，再次使用 Alt + 鼠标左键 选择一个位置

这里推荐使用 单点 操作而不是 长按 ，你可以动手试试以查看区别

③
